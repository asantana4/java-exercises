# Inheritance Exercise

## Goal
Practice inheritance in Java.

## Description
Worker and Employee are the parent classes.
Employee extends Worker.
SalariedEmployee and HourlyEmployee extend Employee.
SalariedEmployee makes use of the super method terminate() in its
retire() method.
SalariedEmployee and HourlyEmployee override collectPay().


## Concepts
- inheritance
- constructor chaining
- encapsulation and access modifiers
- method overriding
- polymorphism
